What is FALO Survey System?

FALO Survey System 是一個 **AI Friendly 問卷系統範例**。它利用輕量級的前端 HTML 結合 Google 生態系（Apps Script, Sheet, Drive），建立一個可以直接供 AI 進行深度分析的資料採集管道。

主要目的

  • • 讓初學者理解 **AI Data Pipeline** 的運作邏輯
  • • 提供一個 **可快速模仿與部署** 的完整範例系統
  • • 示範如何從源頭就設計出 **AI 可讀** 的資料格式

核心技術棧

HTML 前端 + Google Apps Script API + Google Sheet 儲存 + Google Drive 附件管理。

Core Features

Anonymous Upload

支援免登入上傳附件至 Drive，打破原生表單限制。

Semantic Record

每一筆紀錄皆包含完整語意上下文，供 AI 極速解析。

Sheet Storage

資料直接落地於 Google Sheet，實現低成本持久化儲存。

Drive Integration

自動建立資料夾結構，將檔案與問卷資料精準對位。

Identity Trio

透過 ID/PIN/Reward Code 組合，完成匿名身份與兌換機制。

AI Ready Structure

內建三層分隔設計，讓資料無需處理即可直接進行分析。

為什麼不是一般 Google Forms？

Google Forms

這是一套成熟的 **資料收集工具**。適合一般快速統計，但難以自定義資料儲存的語意結構與匿名檔案管道。

FALO Survey System

這是一個 **AI Data Pipeline 範例**。它示範了從前端到後端如何為了 AI 理解而進行資料優化，是建構 AI 系統的基礎架構。

How to Copy & Build Your Own

本系統的設計初衷是讓每位學員都能 **「模仿式開發」**。透過簡單的三步驟，你就能將這套 AI Data Pipeline 部署為個人專屬的版本。

1
從範例開始 (Start from Example)

老師提供一套 100% 穩定且可運行的範例系統。

2
透過 AI 修改 (AI Driven Mutation)

學員使用 AI 修改 HTML 問卷內容，進行客製化調整。

3
部署你的版本 (Deploy Your Version)

將修改後的代碼與後端 API 連結，生成專屬服務。

AI Friendly Data Encoding

為什麼不直接用 JSON？因為三層分隔具備極佳的 AI 解析經濟性與人類可讀性。

LEVEL 1

,

CSV 欄位分隔

確保資料可直接匯入所有主流資料庫與表格軟體。

LEVEL 2

;;;

子鍵值對區分

區分題目與答案，讓 AI 在解析時能極速精準分割標籤。

LEVEL 3

||

多選答案集合

完整呈現多選答案清單，不破壞 CSV 基礎結構。
